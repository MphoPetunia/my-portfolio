Landing Page project

I wanted to create a clean and user-friendly interface using the trio of HTML, CSS, and JavaScript. The design focused on simplicity, with a well-structured layout and intuitive navigation. I utilized HTML for the page structure, CSS for styling to make it visually appealing, and added some interactive elements with JavaScript to enhance the user experience. Whether it's a subtle hover effect or a responsive design that adapts to different screen sizes, I made sure that every element served a purpose in creating a seamless and engaging front page.

Weather App

For my second project, I took it up a notch and created a dynamic weather app that not only showcased my HTML, CSS, and JavaScript skills but also incorporated a live API to fetch real-time weather data. Users can enter a location, and the app pulls in the current weather conditions, temperature, and other relevant information. I added a touch of interactivity with JavaScript, allowing for seamless updates without requiring a page reload. The design is both functional and aesthetically pleasing, with an intuitive layout that provides users with the information they need at a glance. It's not just a static webpage; it's a practical tool that brings the latest weather information to the fingertips of users in a visually engaging way.
